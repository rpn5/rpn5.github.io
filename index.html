What is Rumi?

Rumi is the outcome of a fundamental realization that the Translytics problem is not just a data problem - it is fundamentally a data+compute problem.

Rumi combines big data storage, stream processing, streaming analytics, real-time data ingestion, live querying and analytics, operational storage and scalable compute in a single product.

Performance

Rumi™ implements several key techniques such as cut-through serialization, off-heap data storage, transaction pipelining, and memory mapped IO to keep performance in the low microsecond range.

Storage

Rumi™ ensures full transactional consistency agent storage is highly recommended for maximum agility and performance. Rumi™ implements private in-memory durable storage per agent. Each agent has a private instance of an in-memory “microDB” with quorum based local persistence and asynchronous connectivity to shared enterprise stores. This enables independent schema management and store administration for maximum performance and agility with data integration to the rest of the enterprise.

Learn More

Messaging & Collaboration

Rumi™'s fundamental abstractions for communications are channels, and these connect agents in a one-to-many fashion. Messages are sent and received as POJOs in a fire-n-forget manner through channels. In addition, Rumi™ implements a messaging provider abstraction layer where one can switch between providers in a plug-n-play manner without any code change. Rumi™ has adapters for many solutions including Solace, Kafka, ActiveMQ, JMS (generic), Rumi™ native messaging offering, and others.

Learn More

Error Handling & Transaction Management

Rumi™ ensures full transactional consistency local to an agent with configurable policies governing how the system should behave in the face of exceptions and failures. Rumi™ relies on guaranteed-eventing and support compensating-transactions. With guaranteed eventing, the platform ensures the fire-n-forget semantic offered to an agent. In the event of a local transaction failure, compensating transactions are dispatched to upstream agents to roll back the changes.

Learn More

System Monitoring

Rumi™ provides a centralized deployment management and monitoring tool. It is a sophisticated tool that allows for viewing of statistics, alerts, and lifecycle events. It also enables agent command-n-control, agent versioning, agent upgrade with zero service loss, trace log consolidation and drill down facilities for troubleshooting and debugging purposes.
